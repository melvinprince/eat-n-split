# Eat-n-Split

## Overview

**Eat-n-Split** is a web application designed to simplify the process of splitting bills among friends or group members. It allows users to input expenses and calculates the amount each person owes, ensuring a fair and transparent distribution of costs.

## Features

- **Add Expenses**: Users can input expenses with details such as amount, description, and payer.
- **Split Calculation**: Automatically calculates the amount each participant owes or is owed.
- **Expense Tracking**: Keeps a record of all expenses and settlements for reference.
- **Responsive Design**: Optimized for various devices, providing a seamless user experience.

## Technologies Used

- **Frontend**: React.js
- **Styling**: CSS

## Installation

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/melvinprince/eat-n-split.git
   cd eat-n-split
   ```

2. **Install Dependencies**:

   ```bash
   npm install
   ```

3. **Start the Application**:

   ```bash
   npm start
   ```

   The application will run on `http://localhost:3000`.

## Usage

1. **Add Participants**: Input the names of all participants involved in the expenses.
2. **Log Expenses**: Enter each expense with the amount, description, and select the payer.
3. **View Balances**: The application will display the amount each person owes or is owed.
4. **Settle Debts**: Keep track of payments made to settle balances.
